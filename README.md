# Business Financial Performance Analysis System

This is a simple Python project that analyzes a business's financial performance based on monthly sales and expense data entered by the user.

The program calculates total sales, total expenses, overall profit, average monthly sales, profit percentage, and identifies the month with the highest sales. It then displays everything in a clear financial report.

## Features

- Calculates total sales and expenses
- Calculates monthly and overall profit
- Finds the best-performing sales month
- Calculates average monthly sales
- Calculates profit percentage
- Generates a simple financial report

## Built With

- Python 3
- Functions
- Lists and Tuples
- Loops
- Conditional Statements
- Basic Financial Calculations
